Cálculo analítico del dominio de una función

Ahora que sabemos qué son el dominio y recorrido de una función y como calcular ambos gráficamente, vamos a estudiar como calcularlos si en lugar de una gráfica de una función tenemos su expresión algebraica (su fórmula).

NOTA: vamos a estudiar analíticamente solo el dominio, ya que para el recorrido hacen falta conceptos que este curso no se conocen.

Ejemplo 1 (ANALÍTICO)

Consideremos la función f(x) = x + 2.

¿Cómo calculamos cuál es su dominio?

El dominio son todos los posibles 'x' que podemos insertarle a la función para que nos devuelva una imagen. Para calcularlo debemos calcular qué valores de 'x' admite la función. Para ello, piensa en valores de 'x' que la función no admita, ¿se te ocurre alguno?

No, en este caso no hay valor de 'x' que la función no admita, lo que significa que admite cualquier valor (cualquier número). Esto se representa como:

Dom f = ℝ

Ejemplo 2 (ANALÍTICO)

Consideremos la función f(x) = x / (x-2)

¿Cómo calculamos cuál es su dominio?

El dominio son todos los posibles 'x' que podemos insertarle a la función para que nos devuelva una imagen. Para calcularlo debemos calcular qué valores de 'x' admite la función. Para ello, piensa en valores de 'x' que la función no admita, ¿se te ocurre alguno?

Veamos, ¿qué es lo que no puede ocurrir en una división?

¡NO PODEMOS DIVIDIR ENTRE 0! Exacto, entonces, los valores de 'x' que esta función no va a admitir son los que hagan que dividamos entre 0, es decir, los que hagan 0 el denominador.

Calculemos entonces la solución de la ecuación x-2 = 0 --> x = 2.

El valor x = 2 es el que hace que dividamos entre 0, ¡por lo que no podemos metérselo a la función! Cualquier otro valor sí que podemos metérselo porque para cualquier otro valor la función devolvería una imagen, pero para x = 2 petaría...

Esto se representa como:

Dom f = ℝ - {2}
